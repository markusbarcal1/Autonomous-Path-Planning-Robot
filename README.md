# Autonomous-Path-Planning-Robot

*Description*



Developed a real-time navigation system that detects obstacles and dynamically determines movement paths using ultrasonic sensor data and servo-based scanning.



*Demo Video:*



Demonstration of real-time obstacle detection and autonomous navigation:

[Watch Video](https://drive.google.com/drive/folders/1s2y5ESJIwO-ihLcFb4FxIxDzyRu50FP8?usp=drive_link)



*Key Features:*



* Ultrasonic distance measurement using echo timing
* Real-time obstacle detection
* Autonomous decision-making algorithm (left/right/U-turn logic)
* Servo-controlled directional scanning
* Dynamic path navigation in obstacle-filled environments



*Tech Used:*



* C (Embedded)
* FRDM-KL46Z
* HC-SR04 Ultrasonic Sensor
* PWM + timers



*Results:*



Successfully navigated obstacle environments using dynamic path selection logic



*What I learned:*



* Converting raw sensor signals into usable data
* Designing real-time decision-making systems
* Integrating sensing, control, and movement into a single system
